# HiveCare Frontend

HiveCare Frontend is a modern React-based web application that enables users to browse home services, create bookings, and manage their accounts through a clean and responsive user interface.

## Features

* User Registration and Login
* Protected Routes
* Service Search and Filtering
* Booking Management
* Responsive Navigation
* React Router Navigation
* REST API Integration with Spring Boot Backend
* Mobile-Friendly UI

## Tech Stack

* React
* React Router
* Axios
* Bootstrap
* CSS3
* JavaScript (ES6+)

## Pages

* Home
* Services
* Login
* Register
* My Bookings
* About Us
* Contact
* Privacy Policy

## Future Enhancements

* User Profile Management
* Booking Status Tracking
* JWT Authentication
* Real-Time Notifications
* Admin Dashboard
* Online Payments
